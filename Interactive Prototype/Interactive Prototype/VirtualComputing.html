Virtual Computing
  • Virtual Computing Lab: The Pitt IT Virtual Lab enables you to access much of the same lab software and resources that you use while sitting in a Pitt IT Student Computing Lab on campus. The convenience of the Virtual Lab is that you can access it from any location, using almost any computer or mobile device, at any time of day.
  • Virtual Desktops for Remote Work: The Pitt IT Virtual Desktop (Microsoft) enables staff to remotely access a set software and resources. The convenience of the Virtual Desktop is that you can access it from any location, using almost any computer or mobile device, at any time of day.
